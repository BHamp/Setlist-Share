# Purpose

This app allows musicians to manage and cooperatively build set lists with bandmates for multiple different bands.

## Requirements

Node v15

## Installation

-Clone or download repo
-cd into setlist-share
-run 'npm install'
-run 'npm start'

## Available Scripts

In the project directory, you can run:
-npm start
-npm run build

## Versions

0.1 - Allow user to add and delete songs to a list that is stored in local storage
